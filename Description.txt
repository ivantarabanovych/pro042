Функціонал фільтрації
1. Фільтрація за типом геометрії — користувач може обрати тип фігури (куб, сфера, циліндр) з випадаючого списку.
2. Фільтрація за кольором — користувач може вибрати колір фігур, щоб відобразити тільки ті фігури, що мають відповідний колір.
3. Фільтрація за розміром — фігури можуть бути відфільтровані за розміром, щоб відображати лише ті, що знаходяться в обраному діапазоні.
4. Комбінована фільтрація — користувач може застосувати декілька фільтрів одночасно.
5. Сортування фігур — можливість сортування фігур за розміром або ім'ям у зростальному чи спадному порядку.
6. Збереження фільтрів — вибрані фільтри зберігаються у localStorage, щоб зберегти налаштування користувача після перезавантаження сторінки.

Використані технології
Three.js — для рендерингу 3D-сцени.
Parcel — для збірки проєкту.
LocalStorage API — для збереження налаштувань фільтрів у браузері.
JavaScript (ES6+) — для створення об’єктів, фільтрації та сортування.
HTML та CSS — для створення інтерфейсу та стилізації.
